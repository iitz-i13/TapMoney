# Usage

起動したら、さっそく使用することができます。

使用した金額を入力するだけでOK!

また、収入で得た金額を入力するだけでOK!

あとは、属性別で目的別のラベルを選ぶだけ！

残高確認画面で、いつでも確認することができます。
